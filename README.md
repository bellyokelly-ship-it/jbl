# 🔥 Jimmy's Big Load (JBL)

> Ultimate Steam Deck optimization plugin for Decky Loader

## Features

| Feature | Description |
|---------|-------------|
| ⚡ PowerShift | Intent-driven TDP control (Performance / Balanced / Battery) |
| 🎮 LSFG Auto-Tune | Frame generation with smart flow rate adjustment |
| 🌡️ Thermal Intel | Real-time thermal management with emergency throttle |
| 🔋 Battery Arc | Session-based battery planning |
| 🧪 Proton Manager | Auto-update GE-Proton |
| 📋 AI Profiles | Per-game profiles with machine learning |
| 🌐 Community Sync | Shared profiles from the community |
| 🥽 XR Support | Viture Luma Ultra + XREAL + Rokid |
| 📊 Analytics | Session tracking and FPS history |
| 🩺 Health Monitor | System diagnostics |

## Installation

1. Install [Decky Loader](https://decky.xyz)
2. Download `jimmys-big-load.zip` from [Releases](https://github.com/Bellyokelly-ship-it/jbl/releases)
3. Extract to `~/homebrew/plugins/`
4. Restart Decky Loader

## LSFG Default
James's personal sweet spot: **2x multiplier @ 50% flow rate** 🎯

## XR Support
Tested with **Viture Luma Ultra** — auto-detects on USB-C connection.
